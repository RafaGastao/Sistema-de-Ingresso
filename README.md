 Sistema de Ingressos para Eventos

Sistema web desenvolvido em Java utilizando Spring Boot para gerenciamento de eventos e compra de ingressos.
   Funcionalidades

- Cadastro de usuários
- Login com autenticação
- Diferenciação de perfil (ADMIN e USUÁRIO)
- Cadastro de eventos (ADMIN)
- Visualização de eventos
- Compra de ingressos
- Cálculo automático do valor (Normal, VIP, Meia)
- Pagamento de ingressos
- Impressão do ingresso
- Controle de estado do ingresso (Reservado, Pago, Utilizado, Cancelado)
